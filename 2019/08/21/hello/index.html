title:如果创建博客

先下载node.js

https://nodejs.org/en/        （这是官网），下载LTS(长期持有)，
安装过后，进入DOS(CMD)，输入node -v,就可以看到版本号，也就是安装成功。

##注册Github
用邮箱或者手机号,进入GitHub右上角，进入your repositories,new一个，
名字必须是比如（aaa.github.io),aaa是你自己的用户名
选公共或私有都行，Skip下面那个选项打钩，然后Create

##下载Git
https://git-scm.com/downloads 下载相应版本的setup
安装过后，进入Git Bash Here，输入ssh-keygen,一直按回车
后面会有（.ssh）文件的保存位置，找到文件，打开id_rsa(PUB文件)复制到github的settings的SSH and GPG KEYS

##下载Hexo
到除C盘以外的盘符，新建一个文件夹，用git进入到文件夹，在文件夹里面鼠标右键，进入git bash here
输入npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo s -p 5555
(可以本地运行)网页里面输入(localhost:5555)

##下载sublime
https://www.sublimetext.com/
安装完成后，打开，把之前新建文件夹的博客文件夹拖入sublime，
打开source/_posts/hello-world.md

打开/* _config.yml，author改成自己

                    url:http://aaa.github.io             (aaa是自己的名字)

滑到文件最下面，type：git
               repo:从github那个新建的文件网站上复制过来

返回git

输入npm install hexo-deployer-git --save
hexo g
hexo d        
//这里要输入GitHub账号和密码，才能上传
